# NBA Stats Analyzer

A Flask web application for analyzing NBA team and player statistics.

## Features

- Team comparison and analysis
- Player performance tracking
- Top performers ranking
- Game-by-game statistics

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser to `http://127.0.0.1:5000`

## Deployment

This app is configured for deployment on Render.com using the `render.yaml` configuration file.
